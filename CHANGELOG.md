# Changelog
All notable changes to this package will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [2.2.0] - 2025-12-16
### Added
- Payment Links SDK now supports UPM.
- `PaymentLinksController` now supports browsing mode for each platform.

### Changed
- Android Platform version updated to v1.3.0.
- iOS Platform version updated to v1.5.0.
- Improved iOS Configuration.

## [2.1.0] - 2025-12-04
### Added
- Added WebGL platform support.
- Configuration file is now generated automatically.

### Improved
- Improved clarity of the configuration file.
- Improved iOS build Post-Process.
